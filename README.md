# Proyecto Base de Datos - 5° Semestre

## 📚 Contexto
Proyecto académico de la materia **Bases de Datos** (5° semestre, FCFM).  
La idea es simular una plataforma similar a **Steam**, enfocada en el modelado y gestión de información.

---

## 🎮 Descripción
La base de datos incluye tablas para manejar:
- **Usuarios** y sus métodos de pago
- **Juegos**, con sus **tags** y **publishers**
- **Logros** y logros obtenidos por usuarios
- **Reviews** y calificaciones
- **Noticias** relacionadas
- **Biblioteca** de juegos adquiridos

---

## 🖥️ Arquitectura
Se usa el patrón **MVC** adaptado a una aplicación en **terminal**:
- **Modelo:** conexión y consultas a MySQL.
- **Controlador:** lógica CRUD y coordinación.
- **Vista:** menú de opciones en consola (cada opción representa una funcionalidad).

---

## 🛠️ Tecnologías
- **Lenguaje:** Python  
- **Base de Datos:** MySQL  
- **Gestión de DB:** Dbeaver y MySQL-Workbench-Community

---

## 🚀 Funcionalidades
- Registro y gestión de usuarios
- Administración de juegos, publishers y tags
- Manejo de biblioteca y compras
- Registro de logros y reseñas
- Menú interactivo en terminal para acceder a cada módulo
